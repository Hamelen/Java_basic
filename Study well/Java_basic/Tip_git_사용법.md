### Git-fork

```

1. Git-fork
   1) 파일 -> init new repository
   2) Project 폴더 선택
   3) gitignore.io (웹사이트)-> OS, 개발언어, IDE 입력 후 생성
   4) 메모장 -> 생성한 내용 붙여넣기(확장자 .gitignore)
   5) 저장할 파일명 : .gitignore 위치 : 프로젝트 폴더 내
   6) 테스트 내용 추가 후 git pork에서 커밋

2. Git-hub 연동
   1) Github에서 프로젝트명과 동일한 이름의 repository 생성
   2) URL 주소 복사
   3) git-fork에서 push

```
