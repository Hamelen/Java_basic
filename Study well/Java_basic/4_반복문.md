### for문
<br>

```

- 동일한 구문들을 여러번 해야 할 경우 사용
>>>> 코드 진행을 상위로 되돌아 갈 수 있는 유일한 방법

1. for
	- 반복 해야할 횟수가 정해져 있을 때 사용

	for(초기식;조건식;증감식){
			조건식이 true일 때 실행
	}

	1) 초기식 진행
	2) 조건식 진행
			- true라면 {   } 실행
				-- 3) 증감식 진행
				-- 2) 로 이동
			- false라면 for문 종료

	이중 for
	for (){
		for(){
		}
	}

	
```

### while문
<br>

```


2. while
	- 반복 해야할 횟수가 정해져 있지 않을 때 사용

 A. while
	while(조건식){
		조건식이 true 일 때 실행
  }

 1) 조건식 실행
		true 라면 {   } 실행
		1) 실행
		2) 다시 조건식으로 이동

	 false라면 while문 종료


B. do - while
	do {

	}while(조건식);

	1) {  } 실행
	2) 조건식 실행
		 true라면 {   } 실행
		 2) 실행

		 false 라면 do - while문 종료


```


### 보조문
<br>

```


3. 보조문
	- 반복문 내에서 사용하는 보조문
	- 다중 반복문 내에서 사용할 때 주의

	1) break
			- for   : 그 즉시 해당 for문 종료
			- while : 그 즉시 해당 while문 종료

	2) continue
			- for   : 그 즉시 증감식으로 이동
			- while : 그 즉시 조건식으로 이동



```


### 종료

```


4. 반복문을 종료 하는 방법
	1) 보조문 break
	2) 조건식을 false 


```
